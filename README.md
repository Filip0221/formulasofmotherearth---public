sklep internetowy z kosmetykami
formulasofmotherearth.com
 to prosty sklep internetowy, który umożliwia użytkownikom składanie zamówień poprzez formularz kontaktowy.
Aplikacja została stworzona z myślą o minimalizmie, prostocie obsługi i łatwej rozbudowie.

Projekt jest obecnie w fazie rozwoju — w przyszłości planowane jest dodanie funkcji takich jak:

system rejestracji i logowania użytkowników,

koszyk zakupowy i obsługa wielu produktów,

integracja z systemami płatności online,

panel zarządzania zamówieniami dla administratora.

Aplikacja została zbudowana w oparciu o nowoczesne technologie:

Backend: Django (Python) – obsługa danych, logika biznesowa, API REST

Frontend: Next.js (React) – interfejs użytkownika, dynamiczne renderowanie i SEO
